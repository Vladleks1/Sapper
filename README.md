# Sapper
A sapper game featuring a dynamic design and a highly customizable GUI. The design changes with every game, with a total of six variations available. It was built using the Pygame and Pygame_menu libraries.
![Image Alt](https://github.com/Vladleks1/Sapper/commits/main/start_menu.png)
You can create a universal field with custom settings.
